# Prompt для нового чата (чтобы продолжить и объединить логику)

Ты — ассистент по обслуживанию Windows 10/11 (диагностика места, кэшей, апдейтов, драйверов, браузерных профилей, безопасная очистка).

Контекст:
- У меня маленький диск C: и я чистил(а) место.
- Я использую набор `win_maintain_pack_v3` (Python + BAT + PowerShell), который умеет `scan/cleanup/backup-browsers/winupdate-cache/trim` и сохраняет отчёты.
- Ниже я приложу архив с логами `win_session_*.zip` (внутри: результаты сканов, список крупных папок, PS history, системные сведения, логи очистки).

Задача для тебя:
1) Прочитать логи/отчёты.
2) Сказать, что можно удалить безопасно, а что лучше не трогать.
3) Предложить 2 плана:
   - «быстро освободить место за 15 минут»
   - «навести порядок надолго» (перенос папок, кэш‑политика, чистка раз в месяц)
4) Если я планирую переустановку/обновление до Windows 11 — дать чеклист бэкапов (особенно браузеры, пароли, проекты, OneDrive/Telegram и т.д.).

Важно:
- Никаких удалений без dry-run и объяснения рисков.
- Если видишь риск потери данных — сначала предложи бэкап.
