
# Руководство пользователя (RU)

## Запуск
Запусти **WinMaintainLauncher.exe**.

## Рекомендуемый сценарий
1) **Audit** — посмотреть “что и где”
2) **Dry-run** — увидеть список планируемых действий без удаления
3) (опц.) **Backup browsers** — если переживаешь за профили/куки
4) **Full cleanup** — выполнить очистку (попросит UAC)
5) **Zip session** — собрать все логи/отчёты в один архив

## Где лежат логи
`%LOCALAPPDATA%\WinMaintain\toolkit\logs\YYYY-MM-DD\`

## Где лежат отчёты
`%LOCALAPPDATA%\WinMaintain\toolkit\out\`

## Советы
- Перед чисткой кэшей закрой браузеры и тяжёлые IDE.
- Если прирост места маленький — смотри AppData/OneDrive/IDE caches.
