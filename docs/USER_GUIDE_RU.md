
# Руководство пользователя (RU)

## Запуск
Запусти **toolkit/START_HERE.bat**. Вверху меню появится строка окружения вида `Python=3.11 | Admin=Yes | TPM=Yes | SecureBoot=No | FreeC=12.5 GB` — по ней видно, что доступно на этой машине.

## Рекомендуемый сценарий
1) **Audit** — посмотреть “что и где”
2) **Quick cleanup** — быстрая чистка кэшей (без риска, не требует прав Админа)
3) (опц.) **Backup browsers** — если переживаешь за профили/куки
4) **Full cleanup** — глубокая очистка (может попросить UAC, трогает более тяжёлые кэши)
5) **Zip session** — собрать все логи/отчёты в один архив

## Где лежат логи
`%LOCALAPPDATA%\WinMaintain\toolkit\logs\YYYY-MM-DD\`

## Где лежат отчёты
`%LOCALAPPDATA%\WinMaintain\toolkit\out\`

## Отчёт об окружении
- JSON: `out\env_report.json`
- Меню **View environment report** открывает файл в Notepad и пытается скопировать его в буфер обмена — удобно отправить в поддержку.

## Советы
- Перед чисткой кэшей закрой браузеры и тяжёлые IDE.
- Если прирост места маленький — смотри AppData/OneDrive/IDE caches.
- Если Python не установлен, пункты меню, связанные с `win_maintain.py`, будут отключены; установи Python 3.x или используй готовый WinMaintainScanner EXE из релиза.
